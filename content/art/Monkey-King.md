This is a player character for the "Monkey King" / "Monking Up!" project, and is based on Sun Wukong. 

I modeled it based on my teammates' sketches, rigged and animated it, as well as set up the animations and interactions in Unity game engine. 

The whole project was created in the span of one semester (several months), but the character was created and implemented within around 3-4 weeks (including learning rigging/animating, as well as fixing up the code for the animations).
