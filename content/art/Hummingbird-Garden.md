This was a game project done for HTW Berlin, DE:HIVE. I created concepts, modeled, rigged and animated the hummingbirds, which were the central point of this VR experience. It was lots of fun grabbing them with my VR hand and see them fall unconcious and turn into ragdolls. Apart from that, I also modeled some environment assets and drew decal textures to place around the location.

First, I created concepts for the birds while researching different kinds of hummingbirds that can be found across certain parts of South America:

::lightbox-images{:items='["kolibri-3.jpg", "kolibri-4.jpg" ]'}
::

::lightbox-images{:items='["kolibri-2.jpg" ]'}
::
The base model was then created and rigged in 3D Software Maya.
::lightbox-images{:items='["kolibri-3d.jpg" ]'}
::
Then, I ensured the animations worked well in the Unity game engine, as well as implemented basic ragdoll feature to allow hummingbirds to be picked up and moved around.
::lightbox-images{:items='["Kolibri-Unity-Test-Flying", "Kolibri-Unity-Test-Ded" ]'}
::

I used particle system to replicate motion blur visible on hummingbird wings moving at a very high speed.
::lightbox-images{:items='[ "Kolibri-Unity-Test-Motion-Blur" ]'}
::

In the final version, hummingbirds had the same base model with different color variations.
::lightbox-images{:items='["kolibri-1.jpg" ]'}
::

The hummingbirds fit well into the environment that surrounds them, while being distinct interactive elements.
::lightbox-images{:items='["kolibri-env-1.png", "kolibri-env-2.png"  ]'}
::

Hummingbirds can be dragged around, pet, fed, some of them even dance, sing or leave a ghost trail in the air.

::
